## Objetivo do software
Sistema desktop para gerenciamento de tarefas, permitindo cadastro de usuários e organização de atividades.

## Funcionalidades do sistema
- Cadastro de usuários
- Login de usuários
- Criação de tarefas
- Listagem de tarefas
- Edição de tarefas
- Exclusão de tarefas
- Filtro por status
